# ai-pkm

Sistema de gestão de conhecimento pessoal operado por IA.

O ai-pkm é a plataforma que sustenta um PKM file-first: aplicação web, BFF, agente IA e camada de automação operando sobre um acervo privado de notas versionado em Git. O conteúdo vive no repositório separado [`pkm`](https://github.com/henricos/pkm) (privado); este repositório contém toda a inteligência, estrutura e comportamento do sistema.

Para entender o projeto, comece por [`docs/overview.md`](docs/overview.md).
