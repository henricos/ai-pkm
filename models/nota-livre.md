# Modelo — Nota Livre

Este modelo se aplica a notas que não se encaixam em nenhum dos modelos estruturados disponíveis (`nota-conceito`, `nota-empresa`, `nota-ferramenta`, `nota-procedimento`). Não impõe seções obrigatórias — o autor define a estrutura conforme o conteúdo pede.

---

## Orientações

- **Título obrigatório:** todo arquivo deve ter um H1 claro que descreva o conteúdo
- **Estrutura livre:** use quantas seções forem necessárias, com os títulos que fizerem sentido para o conteúdo
- **Referências de URL:** quando citar fontes externas, seguir o formato padrão dos demais modelos:
  ```
  [https://url-limpa.com/caminho](https://url-limpa.com/caminho) — descrição resumida do que tem no link
  ```
- **Seção Referências:** incluir ao final quando houver links externos, seguindo o mesmo formato acima

---

## Quando usar

Use `modelo: nota-livre` quando o conteúdo for genuinamente inclassificável nos modelos estruturados — não como saída fácil para evitar escolher o modelo correto. Se o assunto for uma ferramenta, conceito, empresa ou procedimento, prefira o modelo específico.
